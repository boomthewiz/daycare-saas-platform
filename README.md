# daycare-saas-platform
This is the first project I'm building 100% on my own, from scratch. I'm using this project as a way to simultaneously teach myself, and show I can produce a tangible product. The aim of this app is to help daycares and other learning & development centers manage their day-to-day tasks and keep a log of events.

## Features

- Teacher dashboard
- Classroom task management
- Child-level task tracking
- Recurring task templates
- Secure role-based access (RLS)
- Real-time database updates

## Tech Stack

- React / Next.js
- Supabase
- PostgreSQL
- Row Level Security
- Tailwind CSS

## Purpose

This project was built to explore building a real SaaS platform including authentication, database security, and multi-tenant architecture.
